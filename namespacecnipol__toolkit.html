float cnipol_toolkit::ekin ( float  edep,
float  dthick 
)

Parameterized function to convert the measured deposited energy (edep) to the kinetic energy of 12C for a given deadlayer thickness (dthick).

Ekin = p0 + p1*Edep + p2*Edep^2 + p3*Edep^3 + p4*Edep^4

For more details about the energy loss model see Appendix A in http://www4.rcf.bnl.gov/~cnipol/Documentations/Files/NIM_080523.pdf
