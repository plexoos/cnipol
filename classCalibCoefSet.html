Detailed Description

A basic data structure to hold a set of calibration parameters extracted from a detector response curve of a known source of alpha particles.

A gaussian shape is assumed in the case of RHIC polarimeters which is fitted to the data acquired from a single detector channel. See AlphaCalibrator for details.
